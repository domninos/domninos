## ℹ️ Background

When I was a Minecraft PvP player back in 2016, I became fascinated with how server features actually worked behind the scenes.

That curiosity eventually led me to apply for a developer position on a Minecraft server, where I was accepted as a Junior Java Developer.

Over the next few years, I gained experience developing custom server features, gameplay systems, and utilities while learning how large Minecraft communities operate.

In 2020, during the pandemic, I transitioned into freelance development. Freelancing exposed me to a wide variety of client requirements, server architectures, and real-world software challenges that significantly accelerated my growth as a developer.

Although I eventually stepped away from freelancing due to burnout, I never stopped programming. Instead, I focused on improving my skills, learning modern technologies, and continuing to build projects that challenge me technically.

Today, I continue pursuing software engineering while specializing in Minecraft server development, backend systems, APIs, Discord integrations, and full-stack applications.
